Snake Game

I used object-oriented programming as well as python turtle module to build this snake game. It is composed of a moving snake that can be controlled by keyboard (up down left right), it can go to different directions and get food that randomly appears on screen. The snake grows in length as it eats more food. Game will be over either when snake hit its own tail or hit the wall. There is also a scoreboard that keep updating current score. Another highest score board is to keep the highest record of snake game on this device.

Turtle Crossing Game

This game is also built using object-oriented programming and python turtle module. In the game, player controls a turtle that can only go forward. On the screen there are lots of cars that are randomly generated and all of them go horizontally across the screen. Once the player reaches another side of the screen, all the cars will speed up, and the turtle goes back to the initial position ready to cross again. Game will be over once the turtle hits a car. There is also a level board to update current level each time it goes up.

Password Manager Program

It is a program built using TkInter (a GUI package). It can generate a password that contains random amount of letters, numbers as well symbols in random order. It can store the generated password and corresponding website in a Json file. The Email/ Username is pre-set but can be changed. It also has a search function using which user can type in the name of a website, the program will go through json file and show the password by a popup window.

HTML Page
